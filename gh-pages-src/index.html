Features

Everything in one system

Diagen wires together best-in-class open models into a single runnable application.

🎤

Speech-to-Text

Browser-native STT integration captures and transcribes voice input with low latency, streaming results to the dialog pipeline.

🤖

Qwen Text Generation

Runs Qwen2.5-0.5B quantized ONNX model locally via onnxruntime-node—no cloud API required. Conversational context is maintained across turns.

🔊

Neural Text-to-Speech

Webtalk generates expressive speech audio on the server, supporting voice cloning via a reference WAV embedding.

👔

Audio2Face Animation

The audio2afan ONNX model maps audio frames to 52 ARKit blendshape weights, producing lip-sync and facial expression data in real-time.

🌟

VRM 0.0 & 1.0 Avatars

Automatic VRM version detection with expression mapping. Load any compatible VRM avatar model and it will animate with the full blendshape set.

⚡

IPFS Model Distribution

AI models are distributed via IPFS content IDs. The included download script fetches all required model files before first run.

Quickstart

Run in minutes

Clone the repo, download the models, and start the server.

Terminal 
# 1. Clone the repository
git clone https://github.com/AnEntrypoint/diagen.git
cd diagen

# 2. Install dependencies
npm install

# 3. Download AI models via IPFS
node download-models.js

# 4. Start the server
npm start

# Open http://localhost:8080 in your browser
Optional: Voice Cloning 
# Place a reference WAV file to clone a voice
# The server will encode it as a speaker embedding on startup
cp your-voice.wav cleetus.wav
